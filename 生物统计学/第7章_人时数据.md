# 第 7 章 人时数据与生存分析（Person-Time Data & Survival Analysis）

> 对应课件 `7 Person-Time Data.pdf`，覆盖 Rosner 第 14 章。
> 本章是流行病学/临床研究中最关键的部分：当数据带有"**时间**"维度（每人随访时间不同、可能未观察到事件就删失），普通比例就不够用了。

---

## 1. 风险/频率的三种度量

| 度量 | 分子 | 分母 | 适用 |
|---|---|---|---|
| **累积发病率（Incidence Proportion）** | \(0\) 到 \(t\) 的新病例数 | \(t=0\) 时无病人数 | 全员同时随访到 \(t\) |
| **点患病率（Point Prevalence）** | \(t\) 时的当前病例数（含旧病） | \(t\) 时总人口 | 截面调查 |
| **发病率（Incidence Rate / Density）** | \(0\) 到 \(t\) 的新病例数 | **总人时**（如人年） | 队列研究、随访时间不一 |

---

## 2. 发病率（Incidence Rate）

### 2.1 定义
每个个体被随访直到：
1. 发病
2. 死亡
3. 失访（迁出）
4. 研究结束

**总人时** = 所有受试者的随访时间之和（如人年）。

\[
\hat\lambda = \frac{a}{t}
\]

- \(a\)：观察到的事件数
- \(t\)：总人时

### 2.2 例：加工肉摄入与糖尿病

| 摄入组 | 病例数 \(a\) | 人年 \(t\) | 发病率 |
|---|---|---|---|
| 高摄入 | 18 | 2,935 | 0.00613 = 613/10万人年 |
| 不摄入 | 215 | 135,130 | 0.00159 = 159/10万人年 |

> **思考单位**：发病率必须带"每人年"等单位。

---

### 2.3 累积发病率（CumI）

假设**发病率 \(\lambda\) 时间不变**：从泊松过程导出
\[
P(T > t) = e^{-\lambda t}
\]

故
\[
\boxed{\;\mathrm{CumI}(t) = 1 - e^{-\lambda t}\;}
\]

当 \(\lambda t < 0.1\) 时（罕见事件、短随访）：
\[
\mathrm{CumI}(t) \approx \lambda t \quad(\text{Taylor 一阶近似})
\]

> 直觉：低风险时，"累积发病率 ≈ 发病率 × 时间"。

---

## 3. 单样本发病率检验

### 3.1 模型

\(A \sim \text{Poisson}(\mu = \lambda t)\)。检验：
\[
H_0: \lambda = \lambda_0 \quad\text{vs}\quad H_1: \lambda \ne \lambda_0
\]

### 3.2 大样本（正态近似）

\[
\boxed{\;X^2 = \frac{(a - \mu_0)^2}{\mu_0} \sim \chi^2_1\;}
\]

经验法则：\(\mu_0 = \lambda_0 t \ge 10\) 时近似良好。

#### 例：高风险饮食 vs 总体糖尿病发病率

- \(\lambda_0 = 0.004, t=4000 \Rightarrow \mu_0 = 16\)
- 观察 \(a=28\)
- \(X^2 = (28-16)^2/16 = 144/16 = 9.0 > 3.84 = \chi^2_{1,0.95}\)
- → 拒绝 \(H_0\)：高风险组糖尿病发病率显著高于总体。

---

### 3.3 小样本（精确泊松）

\[
p = \min\!\left[2\sum_{k=0}^{a} \frac{e^{-\mu_0}\mu_0^k}{k!},\; 1\right] \quad\text{若 } a < \mu_0
\]

\[
p = \min\!\left[2\!\left(1-\sum_{k=0}^{a-1}\frac{e^{-\mu_0}\mu_0^k}{k!}\right),\; 1\right] \quad\text{若 } a \ge \mu_0
\]

### 3.4 \(\lambda\) 的置信区间

- \(a \ge 10\)：正态近似 \(a \pm z_{1-\alpha/2}\sqrt a\)，再除以 \(t\)
- \(a < 10\)：用泊松分位数

\[
\widehat{\mathrm{CI}}(\lambda) = \left(\frac{c_1}{t},\frac{c_2}{t}\right)
\]

例（高风险饮食）：\(\hat\lambda = 28/4000 = 0.007\)，95% CI：
\[
28 \pm 1.96\sqrt{28} = (17.6, 38.4) \Rightarrow \lambda \in (0.0044, 0.0096)
\]
不含 0.004，与拒绝 \(H_0\) 一致。

---

## 4. 双样本发病率检验

### 4.1 模型转化

两组 \(a_1, t_1\) 和 \(a_2, t_2\)。在 \(H_0:\lambda_1=\lambda_2\) 下：
\[
A_1 \mid (A_1+A_2 = n) \sim \text{Binomial}(n, p_0),\quad p_0 = \frac{t_1}{t_1+t_2}
\]

> 直觉：在 \(H_0\) 下，每个事件落在第 1 组的概率正比于第 1 组的人时占比。

### 4.2 期望与方差（\(H_0\) 下）

\[
E(A_1) = n p_0,\quad \mathrm{Var}(A_1) = n p_0 (1-p_0) = n\frac{t_1 t_2}{(t_1+t_2)^2}
\]

### 4.3 大样本检验（含连续性校正）

要求 \(\mathrm{Var}(A_1) \ge 5\)：
\[
z = \frac{a_1 - E(A_1) \mp 0.5}{\sqrt{\mathrm{Var}(A_1)}} \sim N(0,1)
\]

（\(a_1 > E\) 时减 0.5；\(a_1 \le E\) 时加 0.5）

#### 例：肉摄入与糖尿病

- \(a_1=18\)（高摄入），\(t_1=2935\)；\(a_2=215\)，\(t_2=135130\)
- \(E(A_1) = (18+215)\cdot 2935/138065 \approx 4.95\)
- 等等，按课件数据为 \(a_1=9, a_2=239\) → \(E=5.27, \mathrm{Var}=5.16\)（细节略）
- \(z = (9-5.27-0.5)/\sqrt{5.16} \approx 1.42\)，\(p \approx 0.155\)，**不拒绝**

### 4.4 小样本：精确二项检验

直接用 \(\text{Binomial}(n, p_0)\) 算单/双侧 p 值。当 \(\mathrm{Var}(A_1) < 5\) 必须用此法。

## 5. 发病率比（Incidence Rate Ratio, IRR）

### 5.1 估计

\[
\widehat{\mathrm{IRR}} = \frac{\hat\lambda_1}{\hat\lambda_2} = \frac{a_1/t_1}{a_2/t_2}
\]

### 5.2 \(\ln(\mathrm{IRR})\) 的方差（Delta 方法）

由
\[
\ln(\widehat{\mathrm{IRR}}) = \ln a_1 + \ln t_2 - \ln a_2 - \ln t_1
\]

由 \(A_i \sim \text{Poisson}\Rightarrow \mathrm{Var}(A_i) \approx a_i\)，应用 Delta 法（一阶 Taylor）：
\[
\boxed{\;\mathrm{Var}[\ln(\widehat{\mathrm{IRR}})] \approx \frac{1}{a_1} + \frac{1}{a_2}\;}
\]

### 5.3 95% CI

先在 log 尺度算，再指数变换回来：
\[
c_{L,U} = \ln(\widehat{\mathrm{IRR}}) \mp z_{1-\alpha/2}\sqrt{1/a_1 + 1/a_2}
\]
\[
\widehat{\mathrm{CI}}(\mathrm{IRR}) = (e^{c_L}, e^{c_U})
\]

#### 例

\(a_1=15, t_1=4000\)（暴露），\(a_2=50, t_2=15000\)（非暴露）：
- \(\widehat{\mathrm{IRR}} = (15/4000)/(50/15000) = 0.00375/0.00333 = 1.13\)
- \(\ln(1.13) = 0.122\)
- \(\mathrm{SE} = \sqrt{1/15+1/50} = 0.291\)
- 95% CI for \(\ln\)：\((0.122 - 1.96(0.291), 0.122 + 1.96(0.291)) = (-0.294, 0.538)\)
- 95% CI for IRR：\((e^{-0.294}, e^{0.538}) = (0.75, 1.71)\)（**含 1**，差异不显著）

## 6. 分层人时数据（Stratified Analysis）

### 6.1 为什么分层？

**混杂变量（confounder）**：与暴露、结局都有关。例：年龄既影响是否服药，又影响发病率。直接比较暴露/非暴露会被年龄"混淆"。

→ 按年龄分层，在每层内"公平"比较。

### 6.2 假设：层间 IRR 相同（同质性 Homogeneity）

设第 \(i\) 层（共 \(k\) 层）：
\[
p_{1,i} = a_{1,i}/t_{1,i},\;\; p_{2,i} = a_{2,i}/t_{2,i}
\]

在 \(H_0:\mathrm{IRR}_i = 1\) 下：第 \(i\) 层暴露组事件期望
\[
E_i = (a_{1,i}+a_{2,i})\cdot \frac{t_{1,i}}{t_{1,i}+t_{2,i}}
\]

\[
\mathrm{Var}_i = (a_{1,i}+a_{2,i})\cdot\frac{t_{1,i} t_{2,i}}{(t_{1,i}+t_{2,i})^2}
\]

### 6.3 综合检验统计量

\[
A = \sum a_{1,i},\quad E = \sum E_i,\quad V = \sum \mathrm{Var}_i
\]

\[
\boxed{\;X^2 = \frac{(|A - E| - 0.5)^2}{V} \sim \chi^2_1\;}
\]

需 \(V \ge 5\)。这是**Mantel-Haenszel 风格**的检验。

---

### 6.4 综合 IRR 估计（按方差倒数加权）

\[
\ln(\widehat{\mathrm{IRR}}) = \frac{\sum w_i \ln(\widehat{\mathrm{IRR}}_i)}{\sum w_i},\quad w_i = \frac{1}{\mathrm{Var}\{\ln(\widehat{\mathrm{IRR}}_i)\}} = \frac{1}{1/a_{1,i} + 1/a_{2,i}}
\]

方差最小化 → 这是**最优权重**。

\[
\mathrm{Var}\{\ln(\widehat{\mathrm{IRR}})\} = \frac{1}{\sum w_i}
\]

CI 同上：先 log 尺度做正态 CI，再指数变换。

#### 例：药物 vs 疾病（5 个年龄层）

- 综合 \(\widehat{\mathrm{IRR}} = 1.41\)，95% CI：\((1.17, 1.69)\)
- 比较"未分层"的 crude IRR = 1.25 → 年龄是**负向混杂**（年龄越大用药越少，但发病率越高）

---

### 6.5 同质性检验

若各层 IRR 不同，"综合 IRR"无意义。检验：
\[
H_0:\mathrm{IRR}_1 = \cdots = \mathrm{IRR}_k
\]

\[
X^2 = \sum w_i [\ln(\widehat{\mathrm{IRR}}_i) - \ln(\widehat{\mathrm{IRR}})]^2 \sim \chi^2_{k-1}
\]

> 工作流：**先做同质性检验**；如果不显著（不能拒绝同质），再报告综合 IRR。

---

## 7. 生存数据（Survival Data）

### 7.1 什么是生存数据？

**Time-to-event** 变量：从某个时间起点到事件发生的时间。

定义生存时间需要 3 要素：
1. **明确的事件**（终点）：死亡、复发、出院、确诊等
2. **明确的起点**（启动时钟）：诊断、手术、入组、随机化
3. **时间单位**：天、月、年

**常见生存指标**：

| 指标 | 含义 |
|---|---|
| **OS** Overall Survival | 诊断到任何死因死亡 |
| **PFS** Progression-Free Survival | 治疗开始到疾病进展或死亡 |
| **DOR** Duration of Response | 首次响应（CR/PR）到进展或死亡 |

---

### 7.2 删失（Censoring）

由于时间限制、失访等，**事件可能未观察到**。

**类型**：
- **右删失（Right）**（最常见）：研究结束/失访前未发病 → 真实事件时间 \(\ge\) 观察时间
- **左删失（Left）**：在第一次评估前事件就发生了
- **区间删失（Interval）**：在两次评估之间发生（如两次 MRI 之间出现病灶）

### 7.3 非信息性删失假设

**核心假设**：删失的发生**与未来事件风险无关**——被删失者与仍在风险中的人有同样的未来风险。

**信息性删失**示例（违反假设、产生偏差）：
- 因副作用退出（治疗组的"高风险者"被剔除）
- 因病情严重提前结束随访

观察到的：
\[
T_\text{obs} = \min\{T_\text{true}, T_\text{cens}\},\quad \delta = \mathbf{1}\{T_\text{obs} = T_\text{true}\}
\]

→ 数据由 \((T,\delta)\) 两个变量表示。

> **为什么删失数据有用？** 被在 \(t\) 删失的人 → 真实生存时间 \(\ge t\) → 仍是有效信息。

---

## 8. Kaplan-Meier 估计

### 8.1 KM 基本思路

把 \([0,t)\) 切成若干**不重叠**子区间（在每个观察到的事件时刻断开）。

\[
P(\text{活过 }t) = \prod_{i:y_{(i)}\le t} P(\text{活过第 }i\text{ 段)}
\]

**KM 估计公式**：
\[
\boxed{\;\hat S(t) = \prod_{i:\,y_{(i)} \le t}\!\left(1 - \frac{d_{(i)}}{n_{(i)}}\right)\;}
\]

- \(d_{(i)}\)：在 \(y_{(i)}\) 时刻发生事件的人数
- \(n_{(i)}\)：在 \(y_{(i)}\) **正前一刻**仍在风险集中的人数（既未事件也未删失）

### 8.2 KM 曲线特征

- 始于 1（100% 存活）
- 单调**阶梯下降**（仅在事件时刻下降，删失不下降）
- 若最后一个观察是事件 → 终点降到 0
- 删失个体在自己时刻"安静地"退出风险集（曲线不变化，但通常打小竖标记）

### 8.3 关键量

**\(t\) 年生存概率** = 曲线在 \(t\) 时刻的高度。配 95% CI（**Greenwood 公式**估方差）。

**中位生存时间 \(t_{50}\)**：使 \(\hat S(t_{50}) = 0.5\) 的时间。

> ⚠️ **不要混淆**：
> - "样本中 80 人活过 1 年"——只是观察比例
> - "1 年生存率估计为 80%"——必须用 KM（考虑删失）

---

### 8.4 寿命表（Life Table）

旧方法，用于：
- 所有人观察周期相同
- 数据按固定时间区间（0–1 年、1–2 年）汇总

假设删失在区间内**均匀**分布：
\[
n'_i = n_i - \frac{C_i}{2}
\]

\[
\hat S^*(t) = \prod_{i=1}^{j}\!\left(1 - \frac{d_i}{n'_i}\right)
\]

KM vs 寿命表：
- KM 用**实际**风险集大小；寿命表用**平均**
- KM 间隔由事件时刻决定；寿命表间隔预先固定

---

## 9. Log-rank 检验（比较两组生存曲线）

### 9.1 思路

对每个**未删失事件时刻** \(t_i\)：
- 观察事件数 \(O_{1i}\)（暴露组）
- \(H_0\) 下期望 \(E_{1i} = (O_{1i}+O_{2i})\cdot \dfrac{n_{1i}}{n_{1i}+n_{2i}}\)（按风险集人数比例分配）

把所有事件时刻的 \((O - E)\) 求和，按其方差归一化得到 \(\chi^2\)。

### 9.2 假设与等价名

\[
H_0:S_1(t) = S_2(t)\;\forall t
\]

别称：**Mantel-Cox 检验**，或 Mantel-Haenszel 应用于"在每个事件时刻分层"。

### 9.3 注意事项

1. Log-rank 检验**整条曲线**，不是某一时间点。报告时不要写"6 个月生存率显著不同（log-rank p<0.05）"——应写"两组生存分布显著不同"。
2. 多组：\(H_0: S_1=\cdots=S_k\)。整体显著后，再做两两比较，注意**多重比较修正**。
3. 连续协变量：可分组（三/四分位）做 log-rank；更好用 Cox 回归保留原尺度。
4. **比例风险时功效最高**；曲线交叉时不灵敏。
5. 功效主要由**事件数**决定，不是受试者数。

### 9.4 加权扩展

| 检验 | 强调 |
|---|---|
| Peto, Tarone-Ware | 早期差异 |
| Prentice | 后期差异 |
| Fleming-Harrington | 灵活加权 |

## 10. Cox 比例风险模型

### 10.1 风险函数（Hazard）

\[
\boxed{\;h(t) = \lim_{\Delta t \to 0}\frac{P(t \le T < t+\Delta t \mid T \ge t)}{\Delta t}\;}
\]

含义："在 \(t\) 时仍存活的人，下一刻发生事件的瞬时速率"。

注意：
- 是**速率**（rate），可大于 1
- 是**条件**（仅对仍在风险中的人）

### 10.2 风险函数与生存函数

\[
h(t) = -\frac{d}{dt}\ln S(t),\quad S(t) = \exp\!\left(-\int_0^t h(u)\,du\right)
\]

→ 高风险 ↔ 低生存（一一对应）。

**指数分布特例**：\(h(t)=\lambda\) 常数 → \(S(t) = e^{-\lambda t}\)。

---

### 10.3 Cox PH 模型

\[
\boxed{\;h(t \mid X) = h_0(t)\,\exp(\beta_1 X_1 + \cdots + \beta_p X_p)\;}
\]

- **\(h_0(t)\)**：基线风险函数（所有 \(X=0\) 时的风险），**与时间有关**但**形态不指定**
- **指数部分**：协变量乘性效应
- **没有截距 \(\beta_0\)**（被吸收进 \(h_0\)）

取对数：
\[
\ln h(t\mid X) = \ln h_0(t) + \beta_1 X_1 + \cdots + \beta_p X_p
\]

---

### 10.4 系数解释：风险比（HR）

控制其他变量不变，\(X_1\) 加 1 个单位：
\[
\frac{h(t\mid X_1=x+1)}{h(t\mid X_1=x)} = e^{\beta_1} = \mathrm{HR}
\]

| \(\beta\) | HR | 含义 |
|---|---|---|
| \(>0\) | \(>1\) | 风险升高 |
| \(=0\) | \(=1\) | 无关 |
| \(<0\) | \(<1\) | 保护性 |

**典型检验**：\(H_0:\mathrm{HR}=1\)（即 \(\beta=0\)）。

---

### 10.5 比例风险假设（Proportional Hazards Assumption）

\[
\frac{h(t\mid X_1=x+1,\dots)}{h(t\mid X_1=x,\dots)} = e^{\beta_1}
\]

→ **HR 不随时间变化**（即使 \(h(t)\) 本身随时间变）。

诊断：
- 多组 KM 曲线**大致平行**
- log[-log S(t)] vs log(t) 应大致平行
- Schoenfeld 残差检验

PH 假设违反时：
- 加入时间-协变量交互
- 分层 Cox（不同基线 \(h_0\)）
- 使用 Accelerated Failure Time (AFT) 模型

---

### 10.6 半参数模型

- **参数部分**：\(\beta\)（协变量效应）
- **非参数部分**：\(h_0(t)\)（基线，形态不限）

**估计**：
- \(\beta\)：**偏似然（partial likelihood）**最大化（Cox 1972）——不依赖 \(h_0(t)\)！
- \(h_0(t)\)：仅在需要预测绝对风险时估计（如 Breslow 估计）

> **优雅的地方**：在不假设基线分布的前提下，仍能精确估计 \(\beta\)。

---

## 11. 生存研究的样本量与设计

### 11.1 关键事实

> **统计功效由事件数决定，而不是受试者数！**

- 大量删失会"浪费"样本量
- 招募时间长 + 随访短 ≠ 招募时间短 + 随访长
- 高发病率（如重症晚期癌）→ 短随访就够；低发病率 → 长随访

---

### 11.2 所需事件数公式

**Schoenfeld（1981）等分配 \(n_1=n_2\)**：
\[
\boxed{\;d = \frac{4(z_{\alpha/2} + z_\beta)^2}{[\ln(\mathrm{HR}^*)]^2}\;}
\]

**Freedman（1982）**：
\[
d = \frac{(z_{\alpha/2}+z_\beta)^2 (1+\mathrm{HR}^*)^2}{(1-\mathrm{HR}^*)^2}
\]

通用：
\[
\frac{1}{d_1} + \frac{1}{d_2} = \frac{[\ln(\mathrm{HR}^*)]^2}{(z_{\alpha/2}+z_\beta)^2}
\]

**影响因素**：
- 功效 \(1-\beta\) ↑ → \(d\) ↑
- \(\alpha\) ↓ → \(d\) ↑
- \(|\ln \mathrm{HR}^*|\) ↑（效应越大）→ \(d\) ↓

---

### 11.3 由所需事件数 → 总样本量

\[
N = \frac{d}{P(\text{Event})}
\]

\(P(\text{Event})\) 估计方法：
1. 文献历史值 + 调整研究时长
2. 假设指数生存：\(P(\text{Event by }t) = 1 - e^{-\lambda t}\)
3. Lachin 近似（考虑招募 + 随访）：
\[
P(\text{Event}) = 1 - \frac{1}{6}[\bar S(f) + 4\bar S(0.5a + f) + \bar S(a+f)]
\]
其中 \(a\) 招募期，\(f\) 招募结束后的随访期。

---

### 11.4 由中位生存反推 HR

指数模型下：
\[
S(t_{50}) = 0.5 \Rightarrow \lambda = \frac{\ln 2}{t_{50}}
\]

例：对照组中位 12 月，治疗组 18 月：
\[
\mathrm{HR} = \frac{\lambda_\text{trt}}{\lambda_\text{ctrl}} = \frac{(\ln 2)/18}{(\ln 2)/12} = \frac{12}{18} = 0.67
\]

→ 治疗组风险下降 33%。

---

### 11.5 失访率修正

设 \(P(\text{drop out})\) 为退出比例：
\[
N^* = \frac{d}{P(\text{Event})} \cdot \frac{1}{1 - P(\text{drop out})}
\]

### 11.6 招募率（Accrual rate）

均匀招募：
\[
N = \text{accrual rate} \times a,\qquad a = \frac{N}{\text{accrual rate}}
\]

例：30 事件、\(P(\text{Event})=0.5\) → \(N=60\)；3 人/月 → 招募 \(a=20\) 月；总时长 60 月 → 随访 \(f=40\) 月。

### 11.7 设计工作流

1. **由 HR、power、α 算所需事件数 \(d\)**（Schoenfeld 公式）
2. **固定一项约束，反求另一项**：
   - 固定 \(N\) → 求 \(f\)（随访期）
   - 固定 \(T = a + f\) → 求 \(N = d/P(\text{Event})\)
3. **由招募率算招募期 \(a = N/\text{rate}\)**

## 12. 决策树：生存分析方法选择

```
研究问题
├─ 比较单一组与已知率 → 单样本泊松/正态近似
├─ 比较两组发病率 → 双样本（IRR）
│   ├─ Var(A1) ≥ 5 → 正态近似 + 连续性校正
│   └─ Var(A1) < 5 → 精确二项
├─ 控制混杂变量 → 分层分析 + 同质性检验
├─ 全曲线比较 → Kaplan-Meier + Log-rank 检验
└─ 协变量调整 + 风险比 → Cox PH 回归
```

## 13. 公式速查

| 量 | 公式 |
|---|---|
| 发病率 | \(\hat\lambda = a/t\) |
| 累积发病率 | \(1 - e^{-\lambda t}\)；罕见时 \(\approx \lambda t\) |
| 单样本卡方 | \(X^2 = (a-\mu_0)^2/\mu_0\) |
| IRR | \((a_1/t_1)/(a_2/t_2)\) |
| \(\ln\) IRR 方差 | \(1/a_1 + 1/a_2\) |
| KM 估计 | \(\hat S(t) = \prod (1 - d_i/n_i)\) |
| Cox 模型 | \(h(t\mid X) = h_0(t)\exp(\beta^\top X)\) |
| 风险比 | \(\mathrm{HR} = e^{\beta_j}\) |
| 所需事件数 | \(d = 4(z_{\alpha/2}+z_\beta)^2/[\ln(\mathrm{HR}^*)]^2\) |
